# Flatten

Flattens deep object to a single level.

`
const obj = { a: { b: { c: 1, d: 2 } }, e: 3 };

to

const result = { "a.b.c": 1, "a.b.d": 2, e: 3 };
`

## Install

Just run `npm i` in your terminal :)
